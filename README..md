# Glioma Habitat Analysis

Voxel-wise radiomics habitat analysis for discriminating true progression from pseudoprogression in glioma.

## Features
- Sliding-window radiomics feature extraction
- Multi-modal imaging support
- Gaussian Mixture Model-based habitat clustering
- Comprehensive habitat diversity metrics
- Parallel processing for large datasets
